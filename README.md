# coder_e

coder english 程序员英语学习插件

# 主要功能

1. 查词翻译，仅支持中英文（可加入生词本）
2. 选中文本，自动出现翻译（点击转图标）
3. 配置网页，自动提取单词到生词本
4. 支持生词分组
5. 配置修改
